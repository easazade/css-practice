Gradients

this is the syntax for gradients

        #grad{
        background-image: linear-gradient(direction, stopColor1, stopColor2)
        }

Linear Gradients

1- default direction

2- to right

3- diagonal multi color

4- using angle and transparent



Repeating Gradients

4- repeat with specified percentage for each color



Radial Gradients

4- repeat with specified percentage for each color
