Animations

CSS allows animation of html elements without the use of javascript or flash

follow the steps bellow to create an animation

  • 1-create the @keyframes
  • 2-bind an html element to the keyframes
  • 3-set the other animation properties for the html element like duration, delay, direction etc

we can use negative value for animation-delay. and the animation will start as if it had already been playing for N seconds.

we can set animation-iteration-count for the html element any value we want or we can set it it infinite

animation-direction property values

  • normal - The animation is played as normal (forwards). This is default
  • reverse - The animation is played in reverse direction (backwards)
  • alternate - The animation is played forwards first, then backwards
  • alternate-reverse - The animation is played backwards first, then forwards


animation-fill-mode property

there are 4 values for animation-fill-more property :

  • none
  • forwards
  • backwards
  • both
