Attribute Selectors

we use attribute selectors whenever we want to make a selection in css that has a property with a specific value like : an image with a title that has the word flower in it a link with target value of _blank etc or form input tags with value of password 

  this is an example showing how we style input tags in a form that has a type of text

  input[type=text] {
  width: 100%;
  padding: 12px 20px;
  margin: 8px 0;
  box-sizing: border-box;
}
