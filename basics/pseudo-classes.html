Pseudo-classes

an html element can have different states in when it is rendered in DOM. we do not have know what state it could be at each time but we can set it to behave as we want on those states. these states are defined by pseudo-classes

for example a pseudo class can be used to :

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus
Note: a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective! a:active MUST come after a:hover in the CSS definition in order to be effective! Pseudo-class names are not case-sensitive 



  an example of pseudo classes :
  a.highlight:hover {
  color: #ff0000;
  }



pseudo classes can be used to take an state as an event and plan an action based on that like showing a tool tip when hovering over a div element
